# Database-Learning

This repository is a personal collection of database learning materials, mainly SQL scripts and lab exercises for practicing relational database concepts.

It includes table design, constraints, inserts, queries, and practice files used across different lessons and assignments. Most scripts appear to target SQL Server.

## Repository Layout

- `Lab/` and `lab1/` to `lab10/` contain lab exercises and practice scripts.
- `HCSDL/` contains coursework and assignment SQL files.
- `README.md` explains the purpose of the repository.

## How to Use

Open the `.sql` files in your SQL Server tool of choice and run them as practice or study references. Some scripts create databases and tables, so review them before executing.

## Notes

- This repository is for learning and practice.
- File names may reflect specific class exercises, labs, or assignments.
